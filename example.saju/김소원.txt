김소원 金 璅 湲  ( 구슬 옥 / 흐를 원)
// 옥돌 소 / 흐를. 물이 맑을 원 湲  (수리음양 양호)
// 2: 29 / 24 / 21 / 37, 획수: "8 / 16 / 13"}
// 29: 성공불만(成功不滿)격  여자: 하고 싶은 일을 못하여 불만이 있는 경우가 많음. 쾌활한 성격이나 경솔할 수 있음. /// 명석두뇌 활동력. 대업성취 부귀공명. 사회적지위 예능탁월. 욕망이커서 불만도 큰듯
// 24: 무(無)유(有)창조격   여자: 치밀하고 똑똑함. 돈이 들어가면 나올 줄 모르는 인색함. /// 외유내강 점진발달 자수성가 재성대길운. 외모준수 예능 소질
// 21: 성실, 두령(頭領)격   여자: 질투심이 강하고, 의지가 강하며, 성실하거나 침착하나, 애교가 부족함. /// 삼덕 대길수. 난관헤치며대업완수. 지도자 위치. 두뇌명석 뛰어난 운수
// 37: 권위성실(權威成實)격  여자: 능력 있고, 폭이 넓은 성품으로 운세 좋으나 주변에 남자가 떠나지 않음. /// 의지굳고과단성 어려움 극복 대지대업성취. 행복 부귀영예. 독립성향으로 고립위험
